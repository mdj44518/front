객체(Object)

객체의 기본사용법

  • 객체는 기능(Method)
  • 객체, 메서드()
  • 객체, 속성
  • 객체, 속성=값

내장객체

  • 자바스크립트 엔진에 내장된 객체
  • 문자(String), 날짜(Date), 배열(Array), 수학(Math), 정규표현식(RegExp Object)
  • 내장객체 생성하기
    변수 = new 함수명();
    예) var tv = new Object();
    tv.width = "30cm"; // 속성(정보)
    tv.height = "25cm";
    tv.off = function() {alert("TV off";)} //기능
    alert(tv.width); tv.off();
