제어문

제어문이란?

  • 제어문이란 프로그램의 흐름을 제어할 수 있도록 도와주는 실행문
  • 조건문 : 조건에 따라 실행(if, else, else if)
  • 선택문 : 값이 일치하는 경우 실행 (switch)
  • 반복문 : 반복해서 실행 (while, for)

조건문

  • if(조건식) { 실행문: }
  • 조건식에 논리형 데이터가 아닌 다른형이 오는 경우
    - 0, null, undefined, (비어있는String?) : false로 인식
    - var num=0: if(num) : false로 인식 (숫자'0' 자체를 false로 인식)
  • else문 : if(조건식) { 실행문; } else { 실행문2; }
  • 예제 : 홀짝 구분하기

선택문

  • switch(변수) {
    case 값1 : 실행문1;
    break;
    case 값2 : 실행문2;
    break;
    default : 실행문3;
    마지막엔 default(break 사용안함)
  • 예제 : 즐겨 사용하는 포털은?

반복문

  • while (조건식) { 실행문; 증감식; }
  • 예제 : 1~100 숫자 세기
  • do { 실행문; 증감식; } while (조건식);
  • 예제 : 인사말 반복하기
  • for(초기값; 조건식; 증감식) { 실행문; }
  • break : 반복문을 중단할경우
    continue : 실행문을 넘기고 다음횟차 반복문 실행
  • 예제 : 중첩 for 문,중첩for로 테이블만들기
