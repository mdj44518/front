태그(tag), 요소(Element)

  1. 링크 : a
  2. 문장 : p
  3. 글자단위 : span
  4. 블럭단위 구간설정 : div
  5. 테이블(표) : table
  6. 이미지 : img
  7. 제목 : h1 ~ h6
  8. 리스트 : ul(순서없는 리스트), ol(순서있는), dl(제목,내용으로 구성된 리스트)
  9. 강조 : em, strong
  10. 홈페이지의 머릿말(헤더) : header
  11. 꼬릿말(푸터) : footer
  12. 섹션(콘텐츠의 그룹) : section, article
  13. 측면의 서브 콘텐츠 : aside
  14. 주소(카피라잇,주소,연락처) : address
  15. 입력관련(폼태그) : form, input, textarea, button
  16. 기타 : style, script...
  17. 비표준 : font, b, i, iframe, frameset
