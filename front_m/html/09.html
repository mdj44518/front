CSS 선택자(Selector)

선택자란?

CSS를 적용하는 대상을 불러오는 경로법을 말함.

선택자의 종류

  • 전체선택자(*) : 모든 요소를 선택
  • 요소선택자(태그) : 요소의 기본속성을 지정
  • 아이디선택자(id) : 아이디값으로 호출하여 지정
  • 클래스선택자(class) : 클래스명으로 호출하여 지정
  • 자식 선택자(>) : 바로 아래 단계의 태그만 선택
  • 자손(하위) 선택자( ) : 하위 요소들 모두 선택
  • 가상요소(before, after) : 선택자 앞, 뒤로 콘텐츠를 추가
  • 가상 클래스 : nth-child, checked, focus...
