CSS(Cascading Style Sheets)

CSS란?

  • 넓은 의미로 HTML 문서의 모양을 (디자인) 꾸미는 모든 활동.
  • 좁은 의미로 style태그 안에 작성되는 내용 또는 외부 ***.css의 내용
  • CSS는 웹에서 디자인을 담당.
웹표준 : 기능(html) + 디자인(css) + 구조(JS)의 분리 (면접단골질문)

스타일의 적용방법

  • 인라인 방식 : 태그에서 style 속성을 이용하는 방식
  • 내부스타일 방식 : head태그 내에 style 태그를 이용하는 방식
  • 외부스타일 방식 : head태그 내에 link 태그로 불러오는 방식

스타일상의 주석사용

  • 한줄 주석인 '//'은 사용불가
  • 여러줄 주석인 /* 내용 */ 사용
  • 자바스크립트에서의 주석은 둘 다 사용가능

네이밍 규칙

  • name, id, class등의 작명규칙을 말함.
  • 예약어(태그, 스크립트 용어등 미리 정해진 명령어)는 피하는게 좋음.
  • 첫자가숫자가 올 수 없음.
  • 특수기호(%$#@..)와 space는 사용불가.
  • 낙타표기법(Camelcase), 언더바표기법(Snakecase)
