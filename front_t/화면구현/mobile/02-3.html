반응형웹 (웹테블릿모바일)

  • 반응형 웹이란?

    반응형웹(Responsive web)이란 하나의 웹페이지(마크업)에서 PC, 모바일, 태블릿PC등 접속하는 디스플레이 크기에 따라 레이아웃이 유동적으로 변하도록 만든 웹페이지 접근 기법을 말함.

  • 웹사이트 실제크기

    디바이스의 픽셀 해상도와 실제크기는 같지 않음. 보통 initial-scale이 1이고 width=device-width일 때 제조사에서 정해준 실제크기

  • 핵심은 미디어쿼리

    @media "미디어종류" (조건) { CSS설정 }

  • 디바이스별 크기

    모바일 : 320 ~ 480 : 480px 이하

    테블릿 : 533 ~ 1366 : 1024px 이하

    PC : 1024이상

  • 디바이스의 방향설정

    가로형 : orientation : landscape

    세로형 : orientation : portrait (기본값)

  • 상대크기 단위

    풀화면 단위는 : 100%, 100vh, 100vw

    [상대크기 글자단위]
    - em(px,pt) : 1em은 body가 가진 대문자 M의 폭값, 상속을 받음
    - rem(root em) : 상속을 받지 않고 body의 M의 폭값으로 적용
