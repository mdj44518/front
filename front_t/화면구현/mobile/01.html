FTP(File Transfer Protocol, 파일 전송 프로토콜)

FTP란?

  • 파일을 업로드, 다운로드 하기위한 파일 전송 프로그램을 말함.(알FTP, 파일질라...)
  • 클라이언트가 서버에 파일을 주고 받을 때 사용
  • 클라이언트(client), 로컬(local) : 파일을 다운 받는 측, 주로 서비스를 받는 사용자를 말함.
  • 호스트(host), 서버(server) : 파일이 업로드 되어 있는 측, 주로 서비스를 제공하는 자를 말함.
  • 파일질라 : 화면 왼쪽이 클라이언트, 오른쪽이 서버의 내용이 표현

서버상식

  • 브라우저에 파일명이 없으면 서버는 보통 index.html이 자동으로 불러와 짐.
  • 코드 작성시 외부파일은 상대 경로로 잡는 것이 유리.
  • 서버는 리눅스, 윈도우계열로 구분.
  • 모든 통신은 고유의 포트번호를 갖는다.
    - 인터넷(www, http, https) : 80
    - FTP : 21
    - SSH(보안쉘) : 22
    - telnet : 23
    - pop3 : 110, 메일(전자우편) 가져올 때
    - smtp : 465, 메일을 보낼 때
