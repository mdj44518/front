모바일웹

모바일웹 제작방식

  • 모바일웹과 일반웹(데스크탑)을 분리하여 개발, 예제
  • 가변형(적응형)웹으로 개발 : 고정단위(px) 대신하여 상대단위(%)로 레이아웃 설정, 예제
  • 반응형웹으로 개발 : 디바이스 폭크기에 따른 여러개의 CSS를 적용하는 기술, 예제

모바일 뷰포트 설정

  • mata 태그로 뷰포트설정 : content='width=400'
  • width=400 : 특정크기를 디바이스 폭으로 배율조정
  • width=device-width : 모바일기기의 밀도(dpi)에 따른 비율대로 크기를 지정
  • width=device-width : 모바일기기의 밀도(dpi)에 따른 비율대로 크기를 지정
  • maximum-scale = 6 : 화면확대 최댓값을 설정
  • minimum-scale = 0.5 : 화면확대 최솟값을 설정
  • initial-scale = 2 : 페이지 입장시 기본 확대 비율 설정
