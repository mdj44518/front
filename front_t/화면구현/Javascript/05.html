객체(Object) II

브라우저 객체모델(BOM)

  • 브라우저에 내장된 객체
  • window, screen, location, history, navigator 객체 등이 있다.
  • window 객체
    - open() : 새창열기, 예제
    - alert() : 경고창
    - prompt() : 질의응답창
    - confirm() : 확인/취소 창
    - moveTo() : 브라우저의 위치를 이동/지정, moveTo(x,y)
    - resizeTo() : 브라우저의 크기를 변경/지정, resizeTo(w,h)
    - setInterval()/ clearInterval() : 일정한 시간간격으로 실행문 반복 예제
    - setTimeout()/ clearTimeout() : 일정한 시간이후에 한번만 실행
  • location 객체
    - location.속성;
    - location.메서드()
    - 예제
  • screen 객체
    - 방문자의 모니터 너비, 높이, 가용너비, 가용높이, 컬러수의 정보를 제공
    - screen.width;
    - 예제
  • history 객체
    - 방문기록상의 정보를 활용하는 객체
    - 예제
  • navigator 객체
    - 브라우저의 정보를 제공하는 객체
    - 예제 - 운영체제 및 스크린정보

문서 객체모델(DOM)

  • HTML 문서의 구조를 말함.(Document Object Model)
  • 주된 목적은 자바스크립트로 문서객체를 선택하여 속성, 스타일을 적용하고자.
  • 자바스크립트 돔 개발은 IE8 이하버전과 기타 브라우저간의 호환성이 떨어지는 단점.(제이쿼리는 이러한 호환성을 해결)
  • 선택자
    - 직접선택자, 원거리선택자 : id, class, 폼명, 요소명 선택자
    - 인접관계선택자 : parentNode, childNodes, firstChild, children, previousSibling, nextSibling...
  • 선택자 사용시 주의점
    - 브라우저는 기본적으로 소스의 순서대로 실행하므로 body가 읽힌 이후에 실행을 해야 함.
    - body태그의 마지막 항목에 스크립트를 작성 권장.
    - head에 사용시 window.onload = function() { 스크립트 }
  • 직접선택자로 CSS 적용
    - document.직접선택자("선택자명").style.속성 = 값;
    - document.getElementById("wrap").style.color = #fff;
    - 예제
  • 인접관계선택자로 CSS 적용
    - document.관계선택자("선택자명").style.속성 = 값;
    - document.getElementById("wrap").children[0].style.color = #fff;
    - 예제
