자바스크립트의 기초

변수

  • 변수 선언 : var box / var box=100;
  • 변수의 데이터형 : 문자형, 숫자형, 논리형, null & undefined
  • 네이밍규칙
    • 첫글자는 반드시 영문만
    • 두번째 글자부터는 영문, 숫자, $, _
    • 예약어는 사용불가
    • 변수명은 의미에 맞게 작명
    • 대소문자를 구분한다.

연산자

  • 산술연산자 : + - * / %(나머지)
  • 문자결합 연산자 : t1="문자", t2=999; result=t1 + t2;
  • 대입연산자 : 연산된 값을 변수에 저장할 때
    • a += b -> a = a+b
    • a *= b -> a = a*b
    • a /= b -> a = a/b
    • a %= b -> a = a%b
  • 증감연산자 : a++, a--
  • 비교연산자 : >, <, >=, <=, ==, !=, ===, !==
  • 논리연산자 : ||(또는), &&(그리고), !(아닌)
  • 연산자 우선순위
    1. ()
    2. 단항연산자 : --, ++, !
    3. 산술연산자 : +, -, *, /, %
    4. 비교연산자 : >, <== ....
    5. 논리연산자 : &&, ||
    6. 대입 : =, +=, *=...
  • 삼항연산자, 예)조건식? 참일때 실행문 : 거짓일때 실행문;
