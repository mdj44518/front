이벤트

  • 이벤트란 브라우저에서 방문자가 취하는 모든 동작을 몰함.
  • 마우스 이벤트
    - onmouseover : 마우스 오버시
    - onmouseout : 마우스가 벗어났을 때
    - onmousemove : 마우스가 움직일 때
    - onclick : 클릭시
    - ondbclick : 더블클릭시
  • 키보드 이벤트
    - onkeypress : 지정한 키를 눌렀을 때(기능키, 방향키 제외)
    - onkeydown : 지정한 키를 눌렀을 때(모든키 사용,대소문자 구분x)
    - onkeyup : 키를 (눌렀다) 떼었을 때
  • 기타 이벤트
    - onfocus : 포커스를 얻었을 때
    - onblur : 포커스를 잃었을 때
    - onchange : value 속성값이 바뀌고 포커스가 이동될 때
    - onload : 지정한 요소의 하위요소를 모두 로딩했을 때
    - onunload : 문서를 닫거나 다른 문서로 이동했을 때
    - onsubmit : 폼요소에서 전송을 눌렀을 때;
    - onreset : 폼요소에서 취소 버튼을 눌렀을 때
    - onresize : 크기가 변경되었을 때
    - onerror : 문제가 발생되었을 때
  • 이벤트 기본 사용법
    - 마크업에서 인라인 방식 : onclick="alert('경고');"
    - DOM을 이용한 이벤트 등록 : 선택자.onclick = function() {실행문;}
  • 키보드 접근성
    - 모든 이벤트는 키보드로 사용할 수 있도록 작성.

이벤트 객체

  • 이벤트는 이벤트 핸들러에 의해 이벤트 객체를 생성하여 실행문이 실행된다.
  • 예제 : 이벤트 객체 생성
  • 마우스이벤트 객체 속성
    - clientX : 마우스 이벤트 발생한 X좌표값
    - clientY : Y좌표값
    - pageX : docuemnt(body)의 X좌표값
    - pageY : docuemnt(body)의 Y좌표값
    - screenX : 화면 모니터 X좌표값
    - screenY : 화면 모니터 Y좌표값
    - layerX : position을 적용한 요소의 X좌표값
    - layerY : position을 적용한 요소의 Y좌표값
    - button : 마우스에 눌린 버튼의 값
  • 키보드이벤트 객체 속성
    - keyCode : 눌린 키의 유니코드 값
    - altKey : alt키가 사용되었으면 true, 아니면 false;
    - ctrlKey : ctrl키가 사용되었으면 true, 아니면 false;
    - shiftKey : shift키가 사용되었으면 true, 아니면 false;
  • 전체 이벤트 객체 속성
    - target : this와 같이 이벤트가 발생한 요소를 반환.
