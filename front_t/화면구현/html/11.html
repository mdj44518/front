글자와 배경

글자관련 속성

  • color : 글자색
  • letter-spacing : 글자간격
  • line-height : 줄간격
    두번째줄
  • font-size : 글자크기
  • font-family : 서체변경
  • font-style : 폰트스타일(이탤릭)
  • font-weight : 글자두께
  • font : 폰트관련 대표속성
  • text-align : 문단정렬

배경관련 속성

  • background-color : 배경색
  • background-image : 배경이미지
  • background-size : 배경이미지 사이즈
    - contain :가로/세로중 작은변을 100%로 맞춤
    - cover : 가로/세로중 큰변을 100%로 맞춤
  • background-repeat : 배경이미지 반복설정
  • background-position : 배경이미지 위치 지정(가로정렬, 세로정렬 순으로 작성)
  • background : 배경관련 대표속성
