제이쿼리(jQuery) 기본

제이쿼리란?

  • 모질라사의 자바스크립트 개발자 존 레식 창시.
  • 자바스크립트를 이용해 만든 라이브러리 언어.
  • 자바스크립트로 만들어진 다양한 함수들의 집합.
  • 호환성 문제점 해결 : 크로스 브라우징
  • 쉽고 편리한 애니메이션 효과 기능 구현(다양한 효과지원 메서드 제공)

제이쿼리 라이브러리 연동하기

  • 다운로드 방식 <script type="text/javascript" src="jquery-3.4.1.min.js"></script>
  • 네트워크 전송방식(CDN) <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>

선택자

  • 제이쿼리 선택자 사용시에는 body의 소스가 다운로드된 이후에 사용할 수 있음.
  • 선택자 사용 기본형
    - $("#wrap").css("background-color","red");
    - $("#wrap").attr("alt","배너광고");
  • 직접선택
    - 전체선택자 $("*")
    - 아이디/클래스/요소명 선택자 $("#wrap")
    - 그룹선택자 $("선택자1, 선택자2, 선택자3...")
    - 예제
  • 인접관계 선택자
    - 부모요소 선택자 $("선택자").parent()
    - 상위요소 선택자 $("선택자").parents()
    - 자식요소 선택자 $("선택자 >자식요소")
    - 자식요소들 선택자 $("선택자 자식요소") / $("선택자").children("선택자")
    - 형(이전)요소 선택자 $("선택자").prev()
    - 형(이전)요소들 선택자 $("선택자").prevAll()
    - 지정 형 요소 선택자 $("선택자).prevUntil("요소명")
    - 동생(다음)요소 선택자 $("선택자").next()
    - 동생(다음)요소들 선택자 $("선택자").nextAll()
    - 예제
  • 제이쿼리 탐색 선택자
    - 기존 선택자에서 한번 더 탐색을 해서 선택하는 선택자
    - 위치탐색 선택자 예제
    • $("선택자:first") / $("선택자").first()
    • $("선택자:last") / $("선택자").last()
    • $("선택자:odd") 홀수 인덱스 요소만 선택(짝수요소만 선택)
    • $("선택자:even") 짝수 인덱스 요소만 선택(홀수요소만 선택)
    • $("선택자:nth-child(숫자)") 숫자번째 요소만 선택
    • $("선택자:nth-child(숫자n)") 숫자 배수 요소만 선택
    • $("선택자:nth-last-of-type(숫자)") 뒤에서 부터 숫자 번째 요소만 선택
    • $("선택자:only-child") 부모요소내에 자식요소가 1개인 요소만 선택
    • $("선택자:eq(index)") index 번째 요소 선택
    • $("선택자:gt(index)") index보다 큰 인덱스 요소를 선택
    • $("선택자:lt(index)") index보다 작은 인덱스 요소를 선택
    • $("선택자").slice(index) 선택자중 index부터의 요소를 모두 선택.
    - 속성탐색 선택자 예제
    • $("선택자[속성]") 속성이 포함된 요소만 선택
    • $("선택자[속성=값]") 속성명과 속성값이 같은 요소만 선택
    • $("선택자[속성^=값]") 속성값이 '값'으로 시작하는 요소만 선택
    • $("선택자[속성*=값]") 속성값이 '값'을 포함한요소만 선택
    • $("선택자:hidden") 숨겨져 있는 요소만 선택
    • $("선택자:visible") 보이는 요소만 선택
    • $("선택자:text") type이 text인 input 요소만 선택
    • $("선택자:selected") selected 속성이 적용된 요소만 선택
    • $("선택자:checked") checked 속성이 적용된 요소만 선택
    - 그 외의 선택자 예제
    • $("선택자").contents() 선택자와 가장 가까운 하위요소를 선택, 다른 프레임(iframe)더 접근가능
    • $("선택자:contain(텍스트)") 텍스트를 포함하는 요소만 선택
    • $("선택자:has(요소명)") / $("선택자").has(요소명)요소명을 포함하는 선택자를 선택
    • $("선택자:not(요소명)") 요소명을 제외하여 선택자를 선택
    • $("선택자").filter(요소명) 요소명을 가진 선택자를 선택
    • $("선택자").find(요소명) 선택자 하위(자손)요소 중에서 요소명을 선택
    • $("선택자").end() chain의 흐름상의 이전 선택자
    - 기타 메서드들
    • is() : $("li").is(:visible) : 가상클래스를 가지고 있으면 true, 없으면 false
      is(:checked / selected / visible / hidden)
    • noConflict() : var m=$.noConflict()
      m("#wrap).css("color");
    • get() : $("li").get(0).style.color : 자바스크립트 DOM 선택자 방식을 사용할수 있도록 변환
    • each() : 반복문과 비슷 예제
