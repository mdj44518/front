이벤트 등록 메서드

이벤트 등록

$("선택자").click(function(){ 실행문; });

로딩이벤트

  • load() : 선택한 이미지, 소스가 로딩이 완료된 후 이벤트 발생
  • ready() : 지정한 HTML문서 객체 로딩후 이벤트 발생
  • 브라우저의 렌더링 과정 : DOM생성 > ready() > 콘텐츠표현(로딩완료) > load()

마우스이벤트

  • click() : 요소(선택자)를 클릭했을 때
  • dblclick() : 요소를 더블클릭했을 때
  • mouseout() : 마우스가 선택요소를 벗어났을 때
  • mouseover() : 마우스가 선택요소에 올렸을 때
  • hover() : 마우스가 선택요소에 올렸을/벗어났을 때 각각 발생
  • mousedown() : 마우스 버튼을 눌렀을 때
  • mouseup() : 마우스 버튼을 눌렀다 떼었을 때
  • mouseenter() : 마우스를 올렸을 때(하위요소로는 발생 x)
  • mouseleave() : 마우스를 벗어났을 때(하위요소로는 발생 x)
  • mousemove() : 마우스를 (선택자 위에서)움직였을 때
  • 예제

키보드이벤트

  • keypress() : 키보드를 눌렀을 때, 키보드값을 담고 있음(키보드를 누르는 동안 계속 실행, 기능키 사용X, 대소문자 구분)
  • keydown() : 키보드를 눌렀을 때, 키보드값을 담고 있음(키보드 누르면 한 번만 실행, 모든키 사용가능, 모두 대문자로 인식)
  • keyup() : 키보드를 (눌렀다) 떼었을 때
  • 예제

포커스 이벤트

  • focus() : 선택요소에 포커스를 얻었을 때
  • blur() : 선택요소에 포커스를 잃었을 때
  • focusin() : 선택요소에 포커스를 얻었을 때(부모요소에도 이벤트 발생)
  • focusout() : 선택요소에 포커스를 잃었을 때(부모요소에도 이벤트 발생)
  • 예제

기타 이벤트 메서드

  • scroll() : 스크롤바를 움직였을 때
  • change() : 선택요소(select 또는 value 속성이 있는 요소들)에 값이 변경되었을 때
  • error() : 에러가 발생 했을 때
  • 이벤트가 발생한 요소 추적하기
  • 예제:scroll()
  • 예제:change()

그룹 이벤트 등록 및 삭제

  • 그룹 이벤트란 동일한 선택요소에 한 개이상의 이벤트를 등록하는 것을 말함.
  • on() : $("선택자").on("focus click",function(){실행문});
    - 한 개이상의 이벤트 등록가능
    - 동적요소에도 이벤트가 적용
    - jQuery 1.7이후 부터 사용가능
    - 예제:on()
  • bind() : on()동일하나 동적요소는 적용되지 않음.
  • live() : on()의 구버전, jquery 1.3~1.9 사용가능
  • one() : 한개 이상의 이벤트를 한 번 만 이벤트 발생.
  • off() / unbind() : 이벤트 제거
